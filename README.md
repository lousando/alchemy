# Alchemy ⚗️

File cleaner for video and subtitle files.

## Supported formats:

- Video
  - mkv, webm, mp4, avi, flv
- Subtitle
  - srt, vtt

## Install

- Prerequisites:
  - [Deno](https://deno.com/)
  - ffmpeg
  - mediainfo
  - mkvpropedit

```bash
# global install (recommended)
make install
```

```bash
# standalone binary
make
```
