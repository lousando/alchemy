# Alchemy ⚗️

File cleaner for video and subtitle files.

## Supported formats:

- Video
  - mkv, webm, mp4, avi, flv
- Subtitle
  - srt, vtt

## Install

Prerequisites: 
    - Install [Deno](https://deno.com/)

```bash
# global install (recommended)
make install
```

```bash
# standalone binary
make
```
